# Security Policy

## Supported Versions

Any Python3 version after 3.6 (tested on 3.13),... 

* uv 
* pip 
* Flask
* Flask-SQLAlchemy
* ReportLab
* Psycopg2 or MySQL Connector (depending on your database)
* SQLite3

#### Prereqs

1. uv venv
2. uv self update
3. uv install pip
4. uv pip install flask
5. uv pip install flask_sqlalchemy
6. uv pip install reportlab


## Reporting a Vulnerability

You can submit a vunerability issue by creating in Issues tab.
